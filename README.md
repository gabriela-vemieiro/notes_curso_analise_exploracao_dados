# Trilha de Aprendizado: Análise e Exploração de Dados (EDA)

Este repositório contém uma sequência estruturada de notebooks Jupyter, cobrindo desde a ingestão de dados até testes estatísticos avançados.

## Estrutura da Trilha

### 01. Limpeza e Preparação de Dados (`01_data_cleaning`)
Foco inicial na leitura de diferentes fontes e no tratamento de inconsistências.
*   **Leitura de Dados**: Demonstração e prática sobre como carregar datasets (CSV, Excel, SQL).
*   **Limpeza de Dados**: Técnicas para lidar com valores ausentes (NaN), duplicatas e correção de tipos de dados.

### 02. AED e Engenharia de Atributos (`02_eda_features`)
Exploração visual e criação de novas variáveis para modelos preditivos.
*   **Análise Exploratória de Dados (AED/EDA)**: Identificação de padrões, outliers e correlações usando bibliotecas como Pandas, Matplotlib e Seaborn.
*   **Engenharia de Atributos (Feature Engineering)**: Transformação de variáveis categóricas, normalização, escalonamento e criação de novas features a partir de dados brutos.

### 03. Estatística e Testes de Hipóteses (`03_statistics`)
Validação científica de insights e tomada de decisão baseada em dados.
*   **Teste de Hipóteses**: Aplicação de testes estatísticos (T-test, etc) para validar se as diferenças observadas nos dados são estatisticamente significativas.

---
**Nota:** Todos os notebooks foram revisados para garantir a melhor compreensão dos conceitos teóricos e do código em Python.
